# OBRS Model
### Oncogenically Bounded Regenerative System (OBRS)

## Overview
This project presents a nonlinear dynamical systems framework modeling the balance between **cellular regeneration** and **oncogenic risk** in adult mammalian tissues.

The system explores how regenerative potential is constrained by tumor suppression mechanisms, leading to the emergence of **critical transitions** and **bistable behavior**.

---

## Variables

- **P** – Cellular plasticity  
- **E** – Epigenetic accessibility  
- **O** – Oncogenic load  
- **S** – Immune surveillance  
- **F** – Fibrotic response  

---

## Key Features

- Nonlinear coupled differential equations  
- Emergence of **bistability**  
- Presence of a **critical threshold (tipping point)**  
- Bifurcation behavior controlled by parameter *k*  

---

## Key Scientific Result

The OBRS model exhibits a **bistable regime**, where the system can converge to two fundamentally different outcomes:

- **Regenerative state** → low oncogenic load, high functional plasticity  
- **Malignant state** → high oncogenic load and collapse of regenerative control  

This occurs **for the same parameter values**, depending on initial conditions.

A **critical transition** emerges around a threshold value of *k*, suggesting the existence of **biological tipping points** in tissue regulation.

---

## Results

The system demonstrates:

- Stable regenerative regimes  
- Oncogenic-dominant attractors  
- Bistability dependent on initial conditions  
- Nonlinear phase transitions
