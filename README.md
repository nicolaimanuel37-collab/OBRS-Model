# OBRS Model

Oncogenically Bounded Regenerative System (OBRS)

## Overview
This project explores a dynamical systems model describing the balance between cellular regeneration and oncogenic risk.

The model captures:
- Cellular plasticity (P)
- Epigenetic accessibility (E)
- Oncogenic load (O)
- Immune surveillance (S)
- Fibrotic response (F)

## Key Features
- Non-linear coupled differential equations
- Emergence of bistability
- Critical transition (tipping point)
- Bifurcation behavior depending on parameter k

## Results
The system exhibits:
- Regenerative stable states
- Oncogenic dominant states
- Bistable regime depending on initial conditions
